# Sequence Memory Test

Sequence Memory Test เป็นเกมทดสอบความจำระยะสั้น โดยผู้เล่นต้องจำลำดับช่องที่ระบบแสดง แล้วกดตามลำดับให้ถูกต้อง หากกดผิดเกมจะจบและแสดงผลคะแนน

## ภาพรวมโปรเจค

เกมนี้ถูกออกแบบมาเพื่อใช้เป็นส่วนหนึ่งของระบบทดสอบความสามารถด้านความจำของผู้เล่น โดยสามารถนำผลลัพธ์ เช่น คะแนน ระดับที่ทำได้ ความแม่นยำ และเวลาในการตอบสนอง ไปบันทึกลงฐานข้อมูลได้

## วิธีการเล่น

1. กดปุ่ม Start เพื่อเริ่มเกม
2. ระบบจะแสดงลำดับช่องที่ต้องจำ
3. ผู้เล่นต้องกดช่องตามลำดับที่ระบบแสดง
4. เมื่อผ่านแต่ละ Level จำนวนลำดับจะเพิ่มขึ้น
5. หากกดผิด เกมจะจบและแสดงผลลัพธ์

## ฟีเจอร์หลัก

- แสดงลำดับช่องแบบสุ่มในตาราง 3x3
- เพิ่มระดับความยากตาม Level
- ความเร็วในการแสดงลำดับจะเพิ่มขึ้นตามระดับ
- ตรวจสอบการกดของผู้เล่นแบบ Real-time
- แสดงผล Level ที่ทำได้
- คำนวณคะแนน
- เก็บข้อมูลผลลัพธ์สำหรับส่งไปบันทึก Database

## ข้อมูลที่ระบบเก็บหลังจบเกม

ระบบสามารถส่งผลลัพธ์ผ่าน `onGameComplete` โดยมีข้อมูล เช่น

- playerId
- sessionId
- gameName
- score
- accuracy
- reactionTimesMs
- startedAt
- endedAt
- durationMs
- finalLevel
- mistakes
- correctClicks
- totalClicks
- sequences

## โครงสร้างการทำงานของเกม

### 1. Idle Phase

เป็นหน้าเริ่มต้นของเกม มีปุ่ม Start ให้ผู้เล่นเริ่มเล่น

### 2. Showing Phase

ระบบจะแสดง Pattern หรือ Sequence ให้ผู้เล่นจำทีละช่อง

### 3. Input Phase

ผู้เล่นต้องกดตาม Pattern ที่ระบบแสดงให้ถูกต้อง

### 4. Result Phase

เมื่อผู้เล่นกดผิด ระบบจะแสดงผลลัพธ์ เช่น Level และ Score

## การคำนวณคะแนน

คะแนนคำนวณจากจำนวน Level ที่ผ่าน

```ts
score = passedLevel * 10
```

เช่น ผ่าน 3 Level จะได้คะแนน 30 คะแนน

## การคำนวณความแม่นยำ

```ts
accuracy = correctClicks / totalClicks * 100
```

ระบบจะนำจำนวนครั้งที่กดถูก หารด้วยจำนวนครั้งที่กดทั้งหมด แล้วแปลงเป็นเปอร์เซ็นต์

## การสุ่มลำดับ

เกมใช้ฟังก์ชัน `appendRandomCell()` เพื่อสุ่มช่องใหม่เพิ่มเข้าไปใน Sequence โดยมีเงื่อนไขว่า

- ไม่สุ่มซ้ำกับช่องล่าสุด
- ไม่สุ่มซ้ำกับช่องก่อนหน้า 2 ตำแหน่ง

เพื่อไม่ให้ Pattern ง่ายหรือซ้ำเกินไป

## การเพิ่มความยาก

ทุก ๆ 3 Level ความเร็วของการแสดง Pattern จะเพิ่มขึ้น ทำให้ผู้เล่นต้องจำเร็วขึ้น

```ts
const LEVELS_PER_SPEED_UP = 3
```
