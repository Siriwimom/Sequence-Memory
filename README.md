# Sequence Memory Test Project 1

Sequence Memory Test เป็นเกมทดสอบความจำระยะสั้น โดยผู้เล่นต้องจำลำดับช่องที่ระบบแสดง แล้วกดตามลำดับให้ถูกต้อง หากกดผิดเกมจะจบและแสดงผลคะแนน

## ภาพรวมโปรเจค

เกมนี้ถูกออกแบบมาเพื่อใช้เป็นส่วนหนึ่งของระบบทดสอบความสามารถด้านความจำของผู้เล่น โดยสามารถนำผลลัพธ์ เช่น คะแนน ระดับที่ทำได้ ความแม่นยำ และเวลาในการตอบสนอง ไปบันทึกลงฐานข้อมูลได้

## วิธีการเล่น

1. กดปุ่ม Start เพื่อเริ่มเกม
2. ระบบจะแสดงลำดับช่องที่ต้องจำ
3. ผู้เล่นต้องกดช่องตามลำดับที่ระบบแสดง
4. เมื่อผ่านแต่ละ Level จำนวนลำดับจะเพิ่มขึ้น
5. หากกดผิด เกมจะจบและแสดงผลลัพธ์

## ฟีเจอร์หลัก

- แสดงลำดับช่องแบบสุ่มในตาราง 3x3
- เพิ่มระดับความยากตาม Level
- ความเร็วในการแสดงลำดับจะเพิ่มขึ้นตามระดับ
- ตรวจสอบการกดของผู้เล่นแบบ Real-time
- แสดงผล Level ที่ทำได้
- คำนวณคะแนน
- เก็บข้อมูลผลลัพธ์สำหรับส่งไปบันทึก Database

## ข้อมูลที่ระบบเก็บหลังจบเกม

ระบบสามารถส่งผลลัพธ์ผ่าน `onGameComplete` โดยมีข้อมูล เช่น

### 1. Idle Phase

เป็นหน้าเริ่มต้นของเกม มีปุ่ม Start ให้ผู้เล่นเริ่มเล่น

### 2. Showing Phase

ระบบจะแสดง Pattern หรือ Sequence ให้ผู้เล่นจำทีละช่อง

### 3. Input Phase

ผู้เล่นต้องกดตาม Pattern ที่ระบบแสดงให้ถูกต้อง

### 4. Result Phase

เมื่อผู้เล่นกดผิด ระบบจะแสดงผลลัพธ์ เช่น Level และ Score

## การคำนวณคะแนน

คะแนนคำนวณจากจำนวน Level ที่ผ่าน

```ts
score = passedLevel * 10
```

เช่น ผ่าน 3 Level จะได้คะแนน 30 คะแนน

## การคำนวณความแม่นยำ

```ts
accuracy = correctClicks / totalClicks * 100
```

ระบบจะนำจำนวนครั้งที่กดถูก หารด้วยจำนวนครั้งที่กดทั้งหมด แล้วแปลงเป็นเปอร์เซ็นต์

## การสุ่มลำดับ

เกมใช้ฟังก์ชัน `appendRandomCell()` เพื่อสุ่มช่องใหม่เพิ่มเข้าไปใน Sequence โดยมีเงื่อนไขว่า

- ไม่สุ่มซ้ำกับช่องล่าสุด
- ไม่สุ่มซ้ำกับช่องก่อนหน้า 2 ตำแหน่ง

เพื่อไม่ให้ Pattern ง่ายหรือซ้ำเกินไป

## การเพิ่มความยาก

ทุก ๆ 3 Level ความเร็วของการแสดง Pattern จะเพิ่มขึ้น ทำให้ผู้เล่นต้องจำเร็วขึ้น

```ts
const LEVELS_PER_SPEED_UP = 3
```

# Symbol Matching Test Project 2

Symbol Matching Test เป็นเกมทดสอบความจำระยะสั้นและการจดจำตำแหน่งของรูปทรง โดยผู้เล่นต้องจำตำแหน่งของสัญลักษณ์ที่ระบบแสดง แล้วจับคู่รูปทรงที่เหมือนกันให้ถูกต้อง หากเลือกผิดเกมจะจบทันทีและแสดงผลคะแนน

เกมนี้ถูกออกแบบให้ใช้ระบบ Fair Randomization เพื่อให้ผู้เล่นแต่ละคนได้รับความยากใกล้เคียงกัน แม้รูปแบบการสุ่มจะไม่เหมือนกันก็ตาม

---

## ภาพรวมโปรเจค

เกมนี้ถูกพัฒนาขึ้นเพื่อใช้เป็นส่วนหนึ่งของระบบทดสอบความสามารถด้านความจำและการตอบสนองของผู้เล่น โดยสามารถนำผลลัพธ์ เช่น คะแนน ความแม่นยำ ระดับที่ผ่าน และเวลาในการตอบสนอง ไปบันทึกลงฐานข้อมูลเพื่อใช้วิเคราะห์ข้อมูลในอนาคตได้

ระบบรองรับการใช้งานจริงในระดับ Production และสามารถนำไปใช้ร่วมกับระบบวิเคราะห์ด้าน Cognitive Assessment หรือ Health Assessment ได้

## วิธีการเล่น

1. กดปุ่ม Start เพื่อเริ่มเกม

4. หลังหมดเวลา การ์ดทั้งหมดจะถูกปิด
5. ผู้เล่นต้องกดจับคู่รูปทรงที่เหมือนกันให้ถูกต้อง
6. เมื่อผ่านแต่ละ Level จำนวนคู่ของการ์ดจะเพิ่มขึ้น
7. หากกดผิด เกมจะจบทันทีและแสดงผลลัพธ์

## โครงสร้างการทำงานของเกม

### 1. Idle Phase

---

### 2. Memorize Phase

ระบบจะแสดงการ์ดทั้งหมดให้ผู้เล่นจำตำแหน่งภายในเวลาที่กำหนด

```ts
memorizeTimeMs: 15000
```

ผู้เล่นจะมีเวลา 15 วินาทีในการจดจำตำแหน่งของรูปทรงทั้งหมด

---

### 3. Playing Phase

หลังหมดเวลา การ์ดจะถูกปิด และผู้เล่นต้องกดจับคู่รูปทรงที่เหมือนกันให้ถูกต้อง

หากจับคู่ถูก:
- การ์ดจะถูกเปิดค้างไว้
- ระบบเพิ่มจำนวนคู่ที่ถูกต้อง

หากจับคู่ผิด:
- เกมจะจบทันที
- ระบบแสดง Result Screen

---

### 4. Result Phase

เมื่อเกมจบ ระบบจะแสดงผลลัพธ์ เช่น

- Level ที่ผ่าน
- Score
- Accuracy
- จำนวนคู่ที่จับถูก
- จำนวนครั้งที่กดผิด

---

## การคำนวณคะแนน

คะแนนคำนวณจากจำนวนด่านที่ผ่าน และจำนวนคู่ที่จับถูก

```ts
score = completedLevel * 100 + totalCorrectPairs * 10
```

ตัวอย่าง:

- ผ่าน 3 Level
- จับคู่ถูกทั้งหมด 8 คู่

คะแนนจะเท่ากับ

```ts
score = 3 * 100 + 8 * 10
score = 380
```

---

## การคำนวณ Accuracy

```ts
accuracy = totalCorrectPairs / totalAttempts * 100
```

ระบบจะนำจำนวนคู่ที่จับถูก หารด้วยจำนวนครั้งที่เลือกทั้งหมด แล้วแปลงเป็นเปอร์เซ็นต์

---

## ระบบ Fair Shuffle

เกมนี้ไม่ได้ใช้การสุ่มแบบ Random ปกติ แต่ใช้ระบบ Fair Shuffle Algorithm เพื่อให้ทุกเกมมีความยุติธรรมมากขึ้น

### หลักการทำงาน

- คู่ที่เหมือนกันจะไม่อยู่ติดกัน
- ระยะห่างระหว่างคู่ถูกเฉลี่ยให้ใกล้เคียงกัน
- รูปแบบ Layout จะถูกสุ่มใหม่ทุกเกม
- ใช้ Seed Random ควบคุมการสุ่ม
- ลด Bias ของ Pattern ที่ง่ายเกินไป
