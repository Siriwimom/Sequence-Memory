# Sequence Memory Test Project 1

Sequence Memory Test เป็นเกมทดสอบความจำระยะสั้น โดยผู้เล่นต้องจำลำดับช่องที่ระบบแสดง แล้วกดตามลำดับให้ถูกต้อง หากกดผิดเกมจะจบและแสดงผลคะแนน

เกมนี้ถูกออกแบบให้มีระบบ Fair Sequence Generation เพื่อให้การสุ่มลำดับมีความยุติธรรมมากขึ้น ลด Pattern ที่ง่ายเกินไป และทำให้ผู้เล่นแต่ละคนได้รับความยากใกล้เคียงกัน

---

## ภาพรวมโปรเจค

เกมนี้ถูกพัฒนาขึ้นเพื่อใช้เป็นส่วนหนึ่งของระบบทดสอบความสามารถด้านความจำของผู้เล่น โดยสามารถนำผลลัพธ์ เช่น คะแนน ระดับที่ทำได้ ความแม่นยำ และเวลาในการตอบสนอง ไปบันทึกลงฐานข้อมูลได้

ระบบรองรับการใช้งานจริงในระดับ Production และสามารถนำผลลัพธ์ไปใช้วิเคราะห์ร่วมกับข้อมูลด้านสุขภาพ หรือ Cognitive Assessment ได้ในอนาคต

---

1. กดปุ่ม Start เพื่อเริ่มเกม

6. หากกดผิด เกมจะจบทันทีและแสดงผลลัพธ์

## ระบบ Fair Sequence Generation

เกมนี้ไม่ได้สุ่มแบบ Random ปกติ แต่ใช้ระบบ Fair Sequence Algorithm เพื่อให้ลำดับมีความสมดุลมากขึ้น

---

## หลักการ Fairness ที่ใช้

### 1. Anti-Streak

ป้องกัน cell เดิมซ้ำติดกันมากเกินไป

```ts
maxSameCellStreak: 2
```

เช่น:

❌ ไม่ต้องการ

```text
1 1 1 1
```

✅ อนุญาต

```text
1 1 3 5
```

---

### 2. Balanced Window

ภายใน 5 step ล่าสุด cell เดียวกันจะมีได้ไม่เกิน 2 ครั้ง

```ts
wouldBreakBalancedWindow()
```

ช่วยลด bias ของ cell บางตำแหน่ง

---

### 3. Anti-Bias Shuffle

ระบบจะเลือก cell ที่ถูกใช้น้อยก่อน

```ts
getLeastUsedCandidates()
```

ช่วยให้การกระจายตัวของ cell สมดุลมากขึ้น

---

### 4. Anti-Vertical Pattern

ป้องกันลำดับแนวตั้งตรงคอลัมน์

เช่น

❌ ไม่ต้องการ

```text
1 4 7
```

หรือ

```text
2 5 8
```

ระบบจะตรวจสอบด้วย

```ts
getStraightVerticalRun()
```

---

### 5. Anti-Natural Order

ป้องกัน Pattern เรียงธรรมชาติ

❌ ไม่ต้องการ

```text
1 2 3 4 5
```

หรือ

```text
0 1 2 3 4
```

ระบบจะตรวจสอบด้วย

```ts
wouldCreateNaturalOrder()
```

---

### 6. Local Cluster Protection

ป้องกัน cell กระจุกตัวอยู่ในพื้นที่เดียวกันมากเกินไป

```ts
wouldCreateLocalCluster()
```

ช่วยให้ Sequence กระจายทั่ว Grid มากขึ้น

---

### 7. Row Gap System

ทุก ๆ 2 Level ระบบจะบังคับให้ Sequence กระโดดข้าม Row

```ts
rowGapEveryLevels: 2
```

ช่วยลด Pattern ที่เดาทางง่าย
